O que é arduino

é uma plataforma open-hardware e possui seu próprio ambiente de desenvolvimento baseado na linguagem C, simplificando a programação para iniciantes. O software pode ser encontrado gratuitamente para download neste link, disponível para Mac OS X, Windows e Linux. Esta placa é a versão open-hardware do original, ou seja, possui a mesma estrutura e componentes, mas difere-se pois não foi produzido na Itália.
